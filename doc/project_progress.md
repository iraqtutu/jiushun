# 项目进度 - 2026-03-12

## 已解决的问题
- **工单查询故障**：修复了管理员在工单列表页面输入客户信息无法查询的问题。
  - 原因分析：聚合查询中的正则匹配使用了 `new RegExp`，在某些环境下可能无法正确触发或存在兼容性问题。
  - 解决方案：改用 `dbCmd.regExp` 进行模糊匹配，并增加了忽略大小写的支持。
  - 数据校验：确认了 `create.vue` 提交的数据结构与查询逻辑中的 `customerInfo.name` 等路径完全匹配。
