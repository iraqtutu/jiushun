# 玖顺农机售后服务工单系统 - 用户操作手册 (v1.2)

## 1. 简介
本系统旨在帮助玖顺农机及其合作伙伴（经销商、服务人员）实现售后服务工单的数字化管理。通过微信小程序，您可以轻松创建工单、上传现场照片、查询历史记录，并与厂家保持即时沟通。

## 2. 账号与登录

### 2.1 登录
1. 打开小程序。
2. 允许微信授权登录。
3. 系统将自动识别您的身份。

### 2.2 账号申请
如果您是首次使用：
1. 系统会提示您无权限。
2. 点击跳转到"注册申请"页面中。
3. 填写真实姓名、手机号，选择您的身份（玖顺员工/经销商人员/服务人员），并填写申请备注。
4. 提交后请耐心等待管理员审核。

### 2.3 状态说明
- **审核中**：您的申请正在处理，请稍候再试。
- **已通过**：您可以正常使用系统功能。
- **已拒绝**：请联系管理员或重新提交申请。

## 3. 用户角色

系统支持以下角色，不同角色拥有不同的功能权限：

| 角色 | 说明 |
|------|------|
| admin | 管理员，拥有所有功能的访问权限 |
| 玖顺员工 | 玖顺农机内部员工 |
| 经销商人员 | 经销商合作伙伴 |
| 服务人员 | 售后服务人员 |
| 数据分析员 | 可查看和导出服务单数据 |

## 4. 功能模块

### 4.1 首页
登录成功后，您可以看到首页看板：
- **欢迎信息**：显示您的昵称和当前身份。
- **常用服务**：
  - **新建服务单**：核心功能，用于提交新的服务记录。
  - **我的服务单/服务单管理**：查看历史服务单（管理员/数据分析员可查看所有服务单）。
- **系统功能**（根据角色显示）：
  - **经销商管理**：（仅管理员）维护系统内客户及经销商基础信息。
  - **成员审批**：（仅管理员）审核新用户申请。
  - **用户管理**：（仅管理员）查看并调整所有用户角色。
  - **意见反馈**：提交您对系统的建议或遇到的问题。

### 4.2 新建服务单
新建服务单是系统的核心功能，采用分段式卡片设计，便于填写和查看。

#### 4.2.1 客户资料
- **姓名**：必填，输入客户姓名，支持自动填充历史记录。
- **电话**：必填，输入联系电话。
- **地址**：必填，输入详细地址。
- **农机用途**：必填，选择"自用"或"作业"。
- **经销商**：必填，点击搜索并选择经销商。
- **报修时间**：必填，选择报修日期。

#### 4.2.2 产品信息
- **机器编号**：必填，输入机器编号。
- **发动机号**：必填，输入发动机号。
- **生产日期**：必填，选择生产日期。
- **工作时长**：必填，输入工作时长（小时）。
- **铭牌照片**：必填，上传铭牌照片。
- **产品型号**：必填，输入产品型号。

#### 4.2.3 服务基础信息
- **服务类型**：必填，选择服务类型（三包/维修/保养/技改/其他）。
- **是否收费**：必填，选择"免费"或"收费"。
- **故障分类**：点击选择故障分类，将生成对应的故障卡片。

#### 4.2.4 故障卡片（按分类生成）
每个故障分类会生成一张独立的卡片，包含：
- **故障现象**：必填，详细描述故障表现。
- **处理方法**：必填，详细描述处理过程。
- **更换零件明细**：可新增零件，包含：
  - 零件名称、图号/编码
  - 来源（自带/经销商调拨/同厂调拨/其他）
  - 数量（可加减）
  - 单价和金额
  - 旧件处理方式（带回/丢弃）
- **现场照片**：必填，上传故障、维修后、旧件照片。

故障分类包括：
- 动力系统（发动机、启动马达、发电机、高原模块等）
- 行走、传动系统（前桥、后桥、变速箱、行走传动轴等）
- 液压系统（各类液压油管、HST、齿轮泵等）
- 电气系统（主线束、启动开关、仪表盘等）
- 供油系统（燃油管、燃油箱、燃油滤清器等）
- 插植系统（插植臂、旋转箱、插秧箱等）
- 拉线类（手油门拉线、脚油门拉线等）
- 其他类

#### 4.2.5 附加费用
- **路程费核算**：输入里程和单价，自动计算合计。
- **工时费核算**：输入维修用时（分钟）和单价，自动计算合计和总工时。

#### 4.2.6 最终确认
- **人机合影**：必填，拍摄施工现场及合照。
- **服务完成时间**：必填，选择完成日期和时间。

#### 4.2.7 费用结算概览
系统自动计算并显示：
- 零件费用、总里程、总工时
- 路程费用、工时费用、附加合计（收费时显示）
- 应收合计/物料成本合计

#### 4.2.8 支付方式（仅收费时显示）
- 支持选择：微信支付/支付宝/电汇/现金

#### 4.2.9 草稿功能
如果中途退出，系统会自动保存草稿，下次进入可继续填写。

### 4.3 服务单查询

#### 普通用户（我的服务单）
- 仅展示您自己提交的服务单。
- 可按日期范围筛选。

#### 管理员/数据分析员（服务单管理）
- 展示所有人的服务单。
- 支持高级查询：按客户名、手机号、报单人、产品型号进行模糊搜索。
- 支持日期范围筛选。
- 可导出汇总数据（云端导出，不受分页约束）。

### 4.4 成员审批（仅管理员）
- 处理新用户的入驻申请。
- 可批准或拒绝申请。

### 4.5 经销商管理（仅管理员）
- 维护系统内客户及经销商基础信息。
- 支持搜索和筛选。

### 4.6 用户管理（仅管理员）
- 查看所有用户列表。
- 支持搜索用户名或手机号。
- 可编辑用户信息（姓名、手机号、角色）。
- 可删除用户。

### 4.7 意见反馈
1. 点击首页"意见反馈"。
2. 选择类型（问题反馈/功能建议）。
3. 填写详细内容，并可上传截图。
4. 提交后，管理员可在后台看到并回复。
5. 您可以在列表页看到管理员的回复内容。

### 4.8 用户协议与隐私政策
- 在小程序"我的"页面可查看用户服务协议和隐私政策。

## 5. 常见问题

- **Q: 为什么上传图片很慢？**
  A: 请检查网络状况。系统会自动压缩图片，但在弱网环境下可能需要等待加载提示结束。

- **Q: 提交失败怎么办？**
  A: 请勿关闭小程序，尝试重新点击提交。通常是网络波动导致。

- **Q: 如何修改已提交的服务单？**
  A: 联系管理员，管理员可以在服务单管理中对服务单进行编辑。

- **Q: 草稿会自动保存吗？**
  A: 是的，系统会在您输入内容1秒后自动保存草稿到本地。
