# 玖顺农机售后服务工单系统 - 用户操作手册 (v1.1)

## 1. 简介
本系统旨在帮助玖顺农机及其合作伙伴（经销商、服务人员）实现售后服务工单的数字化管理。通过微信小程序，您可以轻松创建工单、上传现场照片、查询历史记录，并与厂家保持即时沟通。

## 2. 账号与登录

### 2.1 登录
1. 打开小程序。
2. 允许微信授权登录。
3. 系统将自动识别您的身份。

### 2.2 账号申请
如果您是首次使用：
1. 系统会提示您无权限。
2. 点击跳转到“注册申请”页面中。
3. 填写真实姓名、手机号，选择您的身份（玖顺员工/经销商人员/服务人员），并填写申请备注。
4. 提交后请耐心等待管理员审核。

### 2.3 状态说明
- **审核中**：您的申请正在处理，请稍候再试。
- **已通过**：您可以正常使用系统功能。
- **已拒绝**：请联系管理员或重新提交申请。

## 3. 功能模块

### 3.1 首页
登录成功后，您可以看到首页看板：
- **欢迎信息**：显示您的昵称和当前身份，左侧展示公司Logo。
- **功能菜单**：
  - **新建派工单**：➕ 核心功能，用于提交新的服务记录。
  - **我的工单/工单查询**：📄 查看历史工单。
  - **成员审批**：👮 （仅管理员）审核新用户申请。
  - **意见反馈**：💬 提交您对系统的建议或遇到的问题。

### 3.2 新建派工单
1. 点击首页“新建派工单”。
2. **填写信息**：
   - 系统自动生成工单号。
   - 填写客户信息、机器型号、服务内容等。
   - **重要**：如有零件更换，请准确填写零件名称、图号和数量。
3. **上传照片**：
   - 支持拍摄或从相册选择。
   - 必须上传至少一张现场照片。
   - 照片数量应不少于更换零件的数量。
4. **草稿**：如果中途退出，系统会自动保存草稿，下次进入可继续填写。
5. **提交**：确认无误后点击提交，系统将上传图片并保存数据。

### 3.3 工单查询

#### 普通用户（我的工单）
- 仅展示您自己提交的工单。
- 可按日期范围筛选。
- 点击卡片查看详情。

#### 管理员（工单查询）
- 展示所有人的工单。
- 支持按客户名、手机号、报单人、产品型号进行模糊搜索。
- 支持日期范围筛选。

### 3.4 意见反馈
1. 点击首页“意见反馈”。
2. 选择类型（问题反馈/功能建议）。
3. 填写详细内容，并可上传截图。
4. 提交后，管理员可在后台看到并回复。
5. 您可以在列表页看到管理员的回复内容。

## 4. 常见问题
- **Q: 为什么上传图片很慢？**
  A: 请检查网络状况。系统会自动压缩图片，但在弱网环境下可能需要等待加载提示结束。
- **Q: 提交失败怎么办？**
  A: 请勿关闭小程序，尝试重新点击提交。通常是网络波动导致。
